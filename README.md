<p align="center">

New anticheat system created by Japanese

まだα版なので、正常に機能する保証はありません。

## 機能

### 対策するチート
・BedAura/Bedbreak (tanudayo)

・NoSlow (rmatt)

・NoSwing (tanudayo)

・KeepSprint (tanudayo)

・Phase (tanudayo)

・GroundSoof (tanudayo)

・Speed (tanudayo)

・Fly (rmatt)

・GhostHand (tanudayo)

・Timer (tanudayo)

・Blink (tanudayo)

・ViewHP (tanudayo)

## その他の機能
・LagBack

・Alert(sharkac.alerts)

・Ban/Kick(Configで設定可能)
